# NAME

imgsmean - Tool that computes the average of a list of images.

# INSTALLATION AND USAGE

    git clone https://github.com/mauvilsa/imgsmean
    mkdir imgsmean/build
    cd imgsmean/build
    cmake -DCMAKE_INSTALL_PREFIX:PATH=$HOME ..
    make install
    
    imgsmean -h
